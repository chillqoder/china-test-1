# Space Shooter - Next.js 2D Game

A top-down 2D space shooter game built with Next.js (App Router) and PixiJS.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Controls

- **WASD / Arrow Keys** - Move ship
- **Mouse** - Aim
- **Left Click / Space** - Fire weapon
- **Scroll Wheel / 1-2-3** - Switch weapons

## Features

- Parallax starfield with nebulae
- 7 enemy types with unique AI behaviors
- 6 weapon types with distinct visual effects
- Progressive ship damage system
- Wave-based progression with scoring and combos
- Weapon and repair pickups

## Tech Stack

- Next.js 15 (App Router)
- PixiJS 8 (WebGL-accelerated 2D rendering)
- TypeScript
